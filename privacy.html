Datenschutzerklärung

Diese Website erhebt, speichert oder verarbeitet keine personenbezogenen Daten aktiv. Es werden keine Cookies gesetzt, keine Nutzerprofile erstellt und keine Tracking- oder Analyse-Tools eingesetzt. Bitte beachten Sie jedoch, dass beim Laden bestimmter Inhalte (z.B. Karten, Schriftarten oder Skripte von Drittanbietern) technisch bedingt Daten wie Ihre IP-Adresse an externe Server übermittelt werden. Diese Übertragung erfolgt durch Ihren Browser automatisch und ist für die Darstellung der Inhalte erforderlich.

Hosting durch GitHub Pages

Unsere Website wird bei GitHub Pages gehostet, einem Dienst von GitHub, Inc., 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, USA. Beim Aufruf der Seite erhebt GitHub automatisch Informationen wie IP-Adresse, Browsertyp, Betriebssystem, Referrer-URL und Uhrzeit des Zugriffs. Diese Daten dienen der technischen Bereitstellung und Sicherheit der Website.

Mehr dazu: GitHub Privacy Statement

Keine Cookies, kein Tracking

Diese Website setzt keine Cookies, verwendet keine Analysetools und erhebt keine personenbezogenen Daten.

HERE API (Karten- und Routendienste)

Diese Website verwendet Dienste der HERE Global B.V., Kennedyplein 222-226, 5611 ZT Eindhoven, Niederlande. Beim Aufruf der aktuellen Verkehrslage werden IP-Adresse und ggf. Standortdaten an HERE übermittelt. Die Nutzung erfolgt im berechtigten Interesse an der Anzeige geografischer Informationen.

Datenschutz bei HERE: here.com/privacy

basemap.de (Kartendaten öffentlicher Stellen)

Zur Darstellung von Karten verwenden wir basemap.de, ein Angebot der deutschen Landesvermessungsbehörden (AdV). Beim Laden der Kartenkacheln wird Ihre IP-Adresse an die jeweiligen Server (z.B. BKG) übertragen.

Weitere Informationen: basemap.de

Allgemeiner Hinweis zur Datenübermittlung

Beim Laden bestimmter Inhalte (z.B. Karten, Fonts, Skripte) werden Verbindungen zu externen Servern aufgebaut. Dabei kann es zur Übertragung personenbezogener Daten (wie IP-Adressen) an Drittanbieter kommen. Diese Übermittlung erfolgt auf Grundlage von Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse).
